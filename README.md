# Weather App Java  


## Описание
Веб-приложение для предоставления прогноза погоды с AI-сводкой, использующее Java для серверной части.

## Возможности
- **Прогнозы погоды**: Получите подробные прогнозы погоды для любой локации, используя координаты.
- **Поиск городов**: Ищите города для лёгкого нахождения координат.
- **AI-сводки**: Генерация понятных сводок погоды с использованием Google Gemini AI.
- **Отзывчивый веб-интерфейс**: Удобный интерфейс, построенный с использованием шаблонов Thymeleaf.
